## Add your client/student photos here!

### Specifications:
- **Format**: JPG or PNG
- **Size**: 100 x 100 pixels (square)
- **File size**: Under 50KB each
- **Style**: Professional headshots

### How to use:
1. Add photos here with descriptive names (e.g., `sarah-ahmed.jpg`)
2. Update `src/components/Testimonials.jsx` with the file paths

### Example:
```javascript
{
  quote: "Amazing program!",
  name: "Sarah Ahmed",
  role: "Project Manager",
  avatar: '/images/testimonials/sarah-ahmed.jpg',  // Add this
  rating: 5
}
```

If you don't have client photos, the website will show placeholder icons instead (which also looks professional).
