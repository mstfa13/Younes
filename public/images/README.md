# Add your images to this folder

## Structure:

```
images/
├── hero-portrait.jpg        (Recommended: 600x800px)
├── about-photo.jpg          (Recommended: 600x750px)
├── training-program.jpg     (Recommended: 800x600px)
├── companies/
│   ├── vois.png            (Recommended: 200x80px)
│   ├── valeo.png
│   ├── etisalat.png
│   ├── el-sewedy.png
│   └── freelancer.png
└── testimonials/
    ├── client1.jpg          (Recommended: 100x100px)
    └── client2.jpg
```

## Image Optimization Tips:

1. Use WebP format for better compression
2. Compress images before adding them (use TinyPNG, ImageOptim, etc.)
3. Use appropriate dimensions - don't upload massive images
4. Keep file sizes under 500KB for fast loading
